# NBA Shooting Streakiness Analyzer

A Flask dashboard and statistical toolkit investigating whether NBA players shoot more or less streakily in the Playoffs compared to the Regular Season.

## Research Question

Do NBA players exhibit a hot-hand effect (shooting in streaks) under Playoff pressure compared to the Regular Season? We test this using the **Wald-Wolfowitz runs test** combined with a **parametric bootstrap** procedure.

---

## Statistical Methods

### 1. Bootstrap Confidence Intervals — `bootstrap_stats.py`

General-purpose Monte Carlo bootstrap for estimating confidence intervals around arbitrary statistics.

- `Bootstrap` class: resample with replacement, compute per-replicate means, std, and quantiles
- `confidence_interval()`: percentile CIs — two-sided, left-sided (`Lsided`), right-sided (`Rsided`)
- `ci_width_analysis()`: reproduces the seminarium result showing how CI width shrinks as bootstrap sample size grows

### 2. Bootstrap Regression Monte Carlo — `bootstrap_stats.py`

Applies bootstrap resampling to OLS regression coefficients for the `DRIBBLES → TOUCH_TIME` model from `nba_cleaned.csv`.

- `mc_bootstrap()`: B = 10 000 replicates, returns distributions and 95% CIs for intercept b0 and slope b1
- `mse()`: mean squared error — verifies OLS minimises squared error across bootstrap replicates
- `bootstrap_regression_demo()`: runs the full analysis and saves two plots

### 3. Wald-Wolfowitz Runs Test + Parametric Bootstrap — `bootstrap_engine.py`

For each player-game:

1. Compute the **Wald-Wolfowitz Z-score** `Z_i` on the binary shot sequence (1 = made, 0 = missed):

   ```
   E[R_i]   = 2 * n_i * p_i * (1 - p_i) + 1
   Var[R_i] = 4 * n_i * p_i * (1 - p_i) * [1 - 3*p_i*(1-p_i)]
   Z_i      = (R_i - E[R_i]) / sqrt(Var[R_i])
   ```

   Negative Z → fewer runs than chance (streaky). Positive Z → more alternating.

2. **Volume-weight** per-game scores into season-level statistics `Z_RS` and `Z_PO`:

   ```
   Z_ctx = sum(n_i * Z_i) / sum(n_i)
   ```

3. **Test statistic**: `ΔZ = Z_PO − Z_RS`

4. **Parametric bootstrap** (B = 5 000): simulate Bernoulli shots game-by-game using observed `(n_i, p_i)`, recompute `ΔZ*` each rep.

5. **Two-tailed p-value**: `p = mean(|ΔZ*| ≥ |ΔZ|)`

## Installation

```bash
pip install -r requirements.txt
```

---

## Running the Flask App

```bash
python app.py
```

Open `http://localhost:5050` in your browser.

On first load the app reads `shots_cache.csv` if present. Hit **Refresh Data** in the nav (or `POST /refresh`) to pull live data from the NBA Stats API — expect ~3–6 minutes due to rate-limit delays between requests.

## Running Statistical Modules Standalone

```bash
# Bootstrap regression + MSE analysis (requires nba_cleaned.csv)
python bootstrap_stats.py

# Wald-Wolfowitz self-test with synthetic toy data
python bootstrap_engine.py
```
